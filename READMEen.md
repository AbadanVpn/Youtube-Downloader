## 🚀 Install and Run

### Easy Method (Recommended)
- Go to **[Releases](https://github.com/AbadanVpn/YouTube-Downloader/releases)** page on GitHub.
- Download and extract the ready-made **`.zip`** file and run the software.
- **No need to install Python or additional libraries!**

---

### 📱 How to use
- Enter (or paste) the link of the YouTube video, music or playlist.
- Select the save path.
- Select the **Video** or **Audio** mode and the desired quality.
- Click **Start Download** or use **Add to Queue**.
- Use **Schedule** for future downloads.
- **Pause, resume, stop and manage multiple downloads** are all easily accessible
